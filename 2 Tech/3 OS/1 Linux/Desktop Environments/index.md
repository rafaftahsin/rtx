### Manage gnome remote rdp session from command line

```shell
grdctl
```

disable rdp - `grdctl rdp disable`
enable rdp - `grdctl rdp disable`
status - `grdctl status`

Ref: https://manpages.ubuntu.com/manpages/lunar/man1/grdctl.1.html
