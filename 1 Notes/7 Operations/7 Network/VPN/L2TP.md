### Setting up L2TP VPN Client on Ubuntu (Gnome Desktop)

```shell
sudo add-apt-repository ppa:nm-l2tp/network-manager-l2tp
sudo apt-get install network-manager-l2tp network-manager-l2tp-gnome
```
